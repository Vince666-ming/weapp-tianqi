# weapp-tianqi
简单的天气小程序,通过https://pv.sohu.com/cityjson?ie=utf-8
获取ip地址，通过www.tianqiapi.com
获取天气api，需要在这个网站注册获取appid和appsecret。
<br>本来是通过ip.tianqiapi.com获取ip地址，最近发现好像不能用了，所以改为搜狐。
